What does UTF-8 mean?

UTF-8 is a compromise character encoding that can be as compact as ASCII (if the file is just plain English text) but can also contain any unicode characters (with some increase in file size). UTF stands for Unicode Transformation Format. The '8' means it uses 8-bit blocks to represent a character.

What does Deprecated mean?

In the world of software development, "deprecated" refers to functions or elements that are in the process of being replaced by newer ones.

What is a onLoad event?

The onload event occurs when an object has been loaded. onload is most often used within the <body> element to execute a script once a web page has completely loaded all content (including images, script files, CSS files, etc.).

What is SVG?

Scalable Vector Graphics (SVG) is an Extensible Markup Language (XML)-based vector image format for two-dimensional graphics with support for interactivity and animation.

What is a Web App?

What Is a Web-based Mobile App?

An HTML5 app that works like a Web page • A user navigates to a Web page containing the Web-based mobile app

What is the Document Object Model (DOM)?

Allows programs and scripts to update content, structure, and styles on the fly

What is a host process?

It's a run time host in the operating system that controls the overall executive of the browser.

What is an App Package?

It is a file that contains an app's files and folders created for ease of distribution and deployment.

What is an App Container?

It is a separate memory space within a system in which an application runs. It prevents corruption of the operating system if the application fails for some reason and enables a user to cleanly uninstall the app.

What is Name Space?

In computing, a namespace is a set of symbols that are used to identify and refer to objects of various kinds. A namespace ensures that all of a given set of objects have unique names so that they can be easily identified.

What is Windows Runtime (WinRT)?

This is where users run the app and devs can complete development. Made up of layers that provide functionality to Metro style apps and the Windows shell

State Management

State management: is the process of maintaining Web page information during multiple requests for the same or different Web page – eg text fields.
Session state: created when user first requests access to an application; state ends when the user closes the session.
Application state: created when the browser sends the first request for a Web page to the Web server; ends when the user closes the browser.

Web App Storage Concepts

Persistent state information is data that an application needs after the session ends. Many Web applications need to store data (make it persistent) so that users can pick up where they left off when they return to the site.

What do I need in order to test Tests for Touch-enabled Apps?

Debugging & Validating Code

Detecting, finding, and correcting logical or syntactical errors. A syntax error is a typo in the code or a similar error, which is usually revealed during runtime for interpreted apps. A logic error results in the app behaving differently than expected.
